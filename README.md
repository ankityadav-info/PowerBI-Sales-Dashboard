# 📊 Power BI Sales Dashboard

## 🚀 Overview
This repository contains an interactive **Power BI Sales Dashboard** that visualizes sales performance, customer metrics, and market share.  
It includes the **Dashboard (.pbix) file**, **base data**, and **all DAX measures** used in the report.

### 🧭 Steps to Use the Dashboard

#### 1. Open the Power BI file
- Launch **Power BI Desktop**
- Open the file `Sales_Dashboard.pbix`

#### 2. Connect base data
- If Power BI asks for data connections,  
  point them to the `/Data` folder where your Excel/CSV files are stored.

#### 3. Explore & modify
- Interact with visuals, slicers, and KPIs  
- Modify measures or visuals as per your analysis needs
